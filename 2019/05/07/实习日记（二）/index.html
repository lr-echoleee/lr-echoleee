第一周笔记：

  • 行业的头部企业决定了赛道规模，非头部企业就要看其与头部企业的差别？有什么特色店？商业模式能否持续？（VIP陪练与快陪练）

  • 内容电商做硬件？必须得有自己的核心技术（很难）或者品牌（比较难）。例如做线上语文教育，硬件和华为合作，那华为给多少钱就是多少钱，没有主动权的。

  • 若是一个项目短期来看盈利可观，但长期的前景堪忧。或许人民币基金可以考虑短期投资之后卖掉（if有人买），美元基金卖给谁嘞？？不行。（这个不大理解）

  • 针对科学学科的早教，家长没有很大的焦虑和压力感（英语、数学）。

第二周笔记：

  • NLP(Natural Language Processing)：自然语言处理，人工智能（AI）的一个子领域。自然语言是人类智慧的结晶，自然语言处理是人工智能中最为困难的问题之一，而对自然语言处理的研究也是充满魅力和挑战的。NLP-百度百科

  • GMV（Gross Merchandise Volume）：网站成交金额。属于电商平台企业成交类指标，主要指拍下订单的总金额，包含付款和未付款两部分。GMV数值可能远远超过实际成交金额，这也是电商企业（企业）愿意公布GMV数据的原因之一。GMV-MBA智库百科

  • DAU（Daily Active User）：日活跃用户数量。DAU通常统计一日（统计日）之内，登录或使用了某个产品的用户数（去除重复登录的用户），这与流量统计工具里的访客（UV）概念相似。通常DAU会结合MAU（月活跃用户数量）一起使用，这两个指标一般用来衡量服务的用户粘性以及服务的衰退周期。MAU、DAU分别从宏观和微观的角度对服务的用户粘性做了权衡，也可以这么说，MAU更像战略层面的表征，DAU更像战术层面的表征。通过这些宏观和细微的趋势变化，可以对营销及推广提供一些数据支持或者帮助。DAU-百度百科

  • 线上项目主要看融资，线下项目主要看运营（例如在一个地域成功了，投资助其扩张）。

  • 高瓴资本有很多钱，如果投资“资本作为核心壁垒的行业”，就比较容易成功。

  • 最佳的投资：商业模式看好+估值合理；如果估值过高的话就要考虑项目的风险和自己的投资组合分布。

  • 早期的模式（古老的模式）比较容易copy，核心点在创业团队，在员工。

  • 教育产品：提高能力？还是提高了结果？（笔神作文：语文水平？作文成绩？）

第三周笔记：

  • 为什么托育现在比较好做？幼儿园不好做？是否是政策和舆论关注之前的幼儿园？

  • 红英幼儿园做到200家，卖了5个亿。

  • 自己写找出版社出版门槛并不高，出版社邀请门槛高；出书？？创始人可以有同时做很多事情的能力？能力？知名度？

  • 曾经做过什么创业？关键点：做到什么价值？为什么没有做起来嘞？多少钱被收购？多少用户？营收？这个创始人什么角色？股权占比？等。

  • 早托的轻模式不晓得有没有团队在做这个：创业团队做SaaS平台服务，社区里面某妈妈愿意合作，占收入的大比例，那么这个平台帮助环境改造、监控设备、线上课程平台、教育内容等等等等，附近几个妈妈可以把孩子送过来，爷爷奶奶也可以跟过来之类。（滴滴平台？airbnb？）

  • ToC 宣传方法：传销方法（用户自宣传）、
  • ToB 宣传方法：利益、分成（ToB的企业愿不愿意付钱、付不付钱很重要）

  • 例如头部企业市值x，教育行业第一名和第三名估值相差10倍，那么预计我们所投的公司最多能做到多少市值？

  • 分销模式：樊登读书会，在某城市寻找城市合伙人，规模500人樊登就去读书啦，反馈在樊登读书会付费会员提升，收益与城市合伙人分。
  • 凯叔讲故事：IP流量+分销模式

  • SKU（Stock Keeping Unit）：库存量单位。即库存进出计量的基本单元，可以是以件，盒，托盘等为单位。SKU这是对于大型连锁超市DC（配送中心）物流管理的一个必要的方法。现在已经被引申为产品统一编号的简称，每种产品均对应有唯一的SKU号。单品：对一种商品而言，当其品牌、型号、配置、等级、花色、包装容量、单位、生产日期、保质期、用途、价格、产地等属性中任一属性与其他商品存在不同时，可称为一个单品。SKU-百度百科

  • ESOP（Employee Stock Ownership Plans）：员工持股计划。ESOP-百度百科

  • 投融资对接会、峰会之类的举办平台：清科、投中、艾瑞、36氪

第四周笔记：

  • 50%的VC和PE在上海，60%的项目在北京；

  • 分析某个教育项目，例如IT职业培训：
  • 老师的资质要求高？教育的内容更新换代快（是否有数据支持）？学生想学的内容（需求痛点）？不同的学生（应届生？有一定经验的员工？）供需分析。

  • 在存量市场中存活：必须要走的一个路线就是差异化的路线，你提供的服务、你提供的产品、你提供给客户创造的价值必须要有差异，而在差异化大前提进一步逻辑推导下，你会发现其实绕不开的就是细分市场的变化。

  • 职业教育项目：目前做的比较好的中低端是蓝翔、新东方烹饪；高端的就是高顿（ACCA、CFA之类）、中公，客单价高，但客户少、要求高，很难标准化；职业教育的周期很短，考个证考完毕业就需要重新获客，不像K12可以有很长的周期性；目前关注职业教育项目就主要看团队，看获客成本问题。

  • 关于SaaS服务提供商，普适版的SaaS服务的价格比较低，一年也就一两万、两三万的价格，定制版的SaaS服务成本过高；目前的主要瓶颈就是随着B端客户做大，对SaaS服务的定制化要求更高，大客户宁愿自行定制也不大愿意购买提供商的服务，把信息分享给服务提供商，应该如何提高B端企业的客户粘性？这是可以问创业者的问题，不知他们是否有这方面的思考？（ps：作为服务商获得很多B端客户的数据，数据有没有可能有什么增值或者可以发挥的地方？）
  • 教培行业SaaS服务头部企业校宝在线：利润很低，寻求新的增长点，例如和蚂蚁金服合作拓展金融业务；
  • 提供SaaS服务，漏斗状：注册经销商？使用经销商？付费经销商？复购率？

  • 关于一些很细分的赛道的项目，例如艺术类，这种培训很难标准化，而且擅长画画的老师并不一定了解音乐，所以对师资的要求也特别高；而且这类项目一般都只能做到小而美，做不到类似新东方那种很大的规模。针对这种小赛道的项目，主要关注的就是团队的资质？还有关注项目的估值，如果估值一两千万，做得好可以通过并购退出，以一两个亿的估值卖掉，差不多有10倍左右的PE值；如果估值很多，之后就算做得好卖出也没有很高的收益率，就Pass！

  • 目前应该关注的主要项目：天花板足够高的教育项目 & 发展前景很好可用于教育行业的新技术。

  • 天使轮投资更容易做非常紧急的投资决策，一天讨论确定，所以pre突出点、风险提示；

  • 早托红黄蓝直营店利润表5%，K12可以达到15%；
  • 2018年红黄蓝直营店80家，加盟店175家；营收1.56亿美元，平均每家直营店营收187.5万美元，净营收60万美元，净利润10万美元；

  • 单店模型 → 现金流转正3个月，投资回收期24个月；

  • 美元资本结构的公司赴美上市前 → VIE架构：
    可变利益实体，英文表述为Variable Interest Entities（“VIE”），来自于美国会计准则（“US GAAP”）的第46号解释函（“FIN46”），意指投资企业持有具有控制性的利益，但该利益并非来自于多数表决权。常见的VIE架构一般分为境内和境外两个部分：就境外架构而言，一般由创始人在开曼或英属维尔京群岛设立拟上市主体。随后，由该拟上市主体在境外（通常是香港）设立全资子公司，再由该香港子公司在境内设立一家外商独资企业（“WFOE”），至此完成VIE股权控制架构的搭建。就境内架构而言，通常由创始人作为股东设立一家内资企业作为境内运营实体（“VIE公司”），随后通过WFOE与VIE公司及其股东之间签署的一系列协议，使境外拟上市主体能够实现对VIE公司的控制，并达到合并报表的目的。
    • # VIE架构怎么搭？看这一篇就够了！

  • 对投资项目的估值预测：
  • 根据其未来营收、净利润等数据寻找对标企业的估值；判断每一个财务指标是否有异常的状况；challenge；

  • 投资流程：
  • 立项报告 → 签订保密协议拿到财务数据，自己先分析看看合不合理 → 委托第三方进行尽职调查 → 投资各事项商定 → 签协议、打钱

  • 领投方通常可以拿到一个董事席，跟投方估计就是董事会旁听角色？基金公司作为一个单独的投资实体，这个股权不署LP的名；

  • 国内专注教育领域的财务投资基金目前就仅海一资本一家，其余有类似新东方、好未来自己的投资基金，但他们专注的是战略投资，不算财务投资基金；

  • 基金LP的可能情形：想要增加portfolio的多样性，所以将多余的现金委托给基金公司，相当于他们的portfolio就增加了教育方向的投资，理论上来说有助于其对冲其主营业务所在的行业；

  • 基金LP会和基金公司签订投资期协议，例如5个亿十年期，每年收取2.5%的管理费，那么每年5000万*2.5%=75万元的管理费给投资基金，LP会有对这5个亿的预期收益率，这5个亿的利润可以协议按年或者十年后投资期结束后再分，例如十年后产生10个亿的收入，LP分走八成，基金公司分走两成；

  • 对海一资本的美元基金来说，现如今想要投资小额（百万人民以下）的资本给种子轮创业团队，这笔钱可以从海内一资本管理公司中出，也可以问LP借，因为用美元基金投这么小额，产生的手续费用太高了，之后可以转到自己新成立的人民币基金当中；
